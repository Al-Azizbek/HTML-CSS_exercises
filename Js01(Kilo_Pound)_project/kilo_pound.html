For this exercise we will be creating a weight metric convertor

We will take in a number in pounds(libs) and convert it to Kilograms(kg). Right now we only
know enough to do this way, but later on we will how to add logic to this sort problems!

To complete this exercie do the followings:

  • Link this HTML file (Part3_Exercise.htmk) to you own .js file.
  • Use Js to accept a number input pounds(libs).
  • Then in your js file convert this number to kg(* 0.454)
  • Afterwards report back in an alert what the weight is in kg.
  • Then log/write "Conversion completed" to the conlose.
